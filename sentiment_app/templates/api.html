RESTful API Access

Integrate SentiScope's intelligence engine directly into your own applications with a simple JSON API.

POST /analyze

Analyze Text

Send a POST request with a JSON body containing a text field. Returns polarity, subjectivity, and classification results.

cURL Example

# Request curl -X POST http://localhost:5000/analyze \ -H "Content-Type: application/json" \ -d '{"text": "I love building with SentiScope!"}'

Response Schema

// 200 OK · application/json { "sentiment": "POSITIVE", "emoji": "😊", "color_class": "positive", "polarity": 0.6250, "subjectivity": 0.6000, "word_count": 6, "char_count": 36, "processing_time_ms": 1.2 }

Error Responses

// 400 Bad Request · No text { "error": "No text provided" } // 400 Bad Request · Too long { "error": "Text too long. Maximum 5000 characters." }
No API key required < 50ms response time JSON schema v1.0
